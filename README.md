## ✨ 特色功能

### 🎲 随机分组
- 支持2-4人分组模式
- 一键随机分配
- 自定义玩家名单
- 快速重新分组

### 📝 备忘录
- 快速记录
- 简洁界面
- 便捷管理

### 🎨 个性化
- 14款精美图标任意切换
- 5种动态渐变主题：
    - 🌈 彩虹渐变
    - 🌅 日落渐变
    - 🌊 海洋渐变
    - 💫 霓虹渐变
    - 💜 蓝紫渐变
